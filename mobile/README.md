## Quick Start

1. Navigate to your platform directory
2. Choose a sample
3. Follow the sample's README

## Common Setup

All mobile samples require:
- API credentials from [portal.chaosity.cloud](https://portal.chaosity.cloud)
- Platform-specific development environment
- `.env` file with credentials

## Sample Categories

- **Maps** - Display interactive maps on mobile
- **Location** - GPS, geolocation, tracking
- **Store Locator** - Find nearby locations
- **Delivery** - Real-time tracking and updates
