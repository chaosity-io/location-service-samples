## Quick Start

1. Navigate to your framework directory
2. Choose a sample
3. Follow the sample's README

## Common Setup

All backend samples require:
- API credentials from [portal.chaosity.io](https://portal.chaosity.io)
- Framework-specific runtime
- Environment variables configuration

## Sample Categories

- **Authentication** - OAuth2 token management
- **Geocoding** - Server-side address processing
- **Webhooks** - Event handling and notifications
- **Batch Processing** - Bulk operations
