# Web Samples

Web application samples for various JavaScript frameworks.

## Available Frameworks

### [JavaScript (Vanilla)](./javascript-vanilla)
Pure JavaScript examples with no framework dependencies. Perfect for learning the basics or integrating into existing projects.

### [React](./react)
React applications using `@chaosity/location-client-react`. Demonstrates hooks, context, and component patterns.

## Quick Start

1. Navigate to your framework directory
2. Choose a sample
3. Follow the sample's README

## Common Setup

All web samples require:
- Node.js 18+
- API credentials from [portal.chaosity.cloud](https://portal.chaosity.cloud)
- `.env.local` file with credentials

## Sample Categories

- **Maps** - Display interactive maps
- **Address** - Autocomplete, validation, geocoding
- **E-commerce** - Checkout flows, store locators
- **Advanced** - Custom controls, complex interactions
