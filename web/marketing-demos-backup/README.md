# Marketing Demos Backup

This directory contains the original demo components from the marketing site.

These demos have been replaced with GitHub references in the marketing site to avoid duplication and reduce maintenance overhead.

## Demos

- **EcommerceDemo.tsx** - E-commerce address validation demo
- **MapDemo.tsx** - Basic map demo
- **MapDemoAWS.tsx** - AWS map demo
- **MapDemoAWSGeocoder.tsx** - Full-featured map with geocoder and filters
- **NearbyPlacesDemo.tsx** - Nearby places search demo
- **RealEstateDemo.tsx** - Real estate location demo

## Corresponding Samples

These demos are now maintained as standalone samples in the samples directory:

- `nextjs-address-finder` - Address validation and autocomplete
- `nextjs-address-finder-full` - Full-featured address finder with all filters
- `nextjs-map-demo` - Interactive map with filters
